00 About

I'm a cybersecurity researcher passionate about penetration testing and red teaming building hands on skills in vulnerability assessment and adversary simulation while actively working toward becoming an offensive security professional.

What sets my work apart is that I don't just use existing tools I build them. I design and assemble custom wireless security hardware from the ground up, combining embedded systems, RF modules, and open-source firmware into real world testing instruments.

From capturing WPA handshakes on 5GHz to replaying Sub-GHz signals, cloning RFID cards, and probing BLE my hardware tools are built for the kind of testing that goes beyond software alone.

Currently focused on expanding my offensive security research, contributing to the hardware hacking community, and taking on real world engagements.

iOS Penetration Testing
Conducted hands on sessions covering the full iOS attack surface from static analysis of IPA binaries and insecure data storage to runtime manipulation using Frida, certificate pinning bypass, and Objection. Android Penetration Testing
Delivered in depth sessions on Android application security covering APK reverse engineering with jadx, dynamic analysis with Frida, ADB exploitation, exposed components via AndroidManifest, insecure SharedPreferences, and traffic interception. Hardware Security Tools How They Work
Ran a dedicated session breaking down the inner workings of custom built hardware security tools including the ESP32 Marauder, 5GHz Deauther, R4TKN, NRF Box, and the BW16 based 5GHz handshake capture device.
